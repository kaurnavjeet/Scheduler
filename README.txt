- Title: Schedule
- Purpose: Allow user to book appointments for customers with contacts.
           Allow user to add, update, and delete appointments.
           Allow user to add, update, and delete customers.
           Allow user to generate reports of appointments.

- Author: Navjeet Kaur
- Email: nkaur24@wgu.edu
- Version: 1.0
- Date: 2/22/2023

- IntelliJ IDEA 2022.2 (Community Edition)
- JDK: Oracle OpenJDK Version 17.0.6
- JavaFX: JavaFX-SDK-17.0.6
- MySQL Connector: mysql-connector-java-8.0.25

- Directions to run the program:
    1. Open project in IDE.
    2. Add my-sql-connector-java-8.0.25 to project libraries
    3. In VM, delete the previous my-sql-connector and add it again. Do it twice if you have to. I sometimes have to add it twice.
    3. Run the program

- Additional Report - User is able to view how many appointments each contact has by month of current year.
                      User needs to select a contact AND a month in order to filter this report correctly.
                      Appointment count shows the number of appointments.

- Additional Information - By Month Radio Button on Main Page filters appointments by the current month of the current year only.
                           Month Combo Boxes in Reports Page filter appointments by current month of the current year only.
